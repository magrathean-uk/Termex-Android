# Reviewer Notes

App package:

- `com.termex.app`

What the app does:

- Android SSH client with keys, certificates, jump hosts, forwarding, snippets, workplaces, and persistent tmux reconnect.

Review help:

- A live server is not required to review the shell flow.
- On the paywall, use `Try Demo Mode` to open the in-app demo server flow.
- Demo mode disables real live connections and shows a simulated terminal.

- https://magrathean.uk/apps/termex
