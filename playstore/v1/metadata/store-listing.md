## Full description

Termex is an Android SSH client for people who live in terminals.

Connect with passwords, SSH keys, or OpenSSH certificates. Review unknown host keys before first use and confirm changed fingerprints before reconnecting. Work through jump hosts, keep identities strict with IdentitiesOnly, and manage local, remote, and dynamic port forwarding from the app.

Guided onboarding helps you set up a server fast, then saved servers, snippets, and workplaces keep common tasks close. Persistent tmux reconnect helps you return to long-lived shells after relaunch. Biometric lock keeps the app gated when you step away.

Features:

- Password, key, and certificate authentication
- Unknown and changed host-key verification
- Jump host / bastion support
- Local, remote, and dynamic forwarding
- SSH config import
- Snippets and workplaces
- Persistent tmux reconnect
- Biometric app lock
- Extra terminal keys, themes, and transcript tools

Termex stores secrets on-device and keeps terminal session restore local to the device.
