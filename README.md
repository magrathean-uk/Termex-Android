# Termex-Android

Android app for Termex.

## Requirements

- Android Studio (latest stable)
- JDK 17
- Android SDK installed via Android Studio

## Setup

1. Open the project in Android Studio.
2. Let Gradle sync finish.
3. Ensure the Android SDK path is configured (Android Studio handles this).

## Build

- `./gradlew assembleDebug`

## Run

- Use Android Studio's Run button for a connected device or emulator.

## Tests

- `./gradlew test`
